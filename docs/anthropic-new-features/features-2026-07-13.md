# Anthropic & Claude API — What's New (2026-07-13)

## Model & API Changes
- Claude Sonnet 5 launched with a 1M token context window, 128k max output tokens, adaptive thinking on by default (manual extended thinking removed), and a new tokenizer producing ~30% more tokens for the same text.
- Fast mode for Claude Opus 4.6 was removed; requests with `speed: "fast"` now silently run at standard speed and pricing instead of erroring.
- Rate limits were raised across the API, with Sonnet and Haiku now matching Opus limits at every usage tier, and tiers consolidated into three (Start, Build, Scale).
- Fast mode for Claude Opus 4.7 is deprecated and will be removed, after which `speed: "fast"` requests will return an error.
- The code execution, web search, and web fetch tools gained new versions adding REPL state persistence, execution time-limit disclosure, and a `response_inclusion` parameter for agentic workflows.
- Claude Sonnet 4 and Claude Opus 4 were retired; all requests to those model IDs now error.
- Claude Fable 5 and Claude Mythos 5 launched with a 1M token context window, always-on adaptive thinking, safety-classifier refusals (`stop_reason: "refusal"`), and an opt-in fallback parameter to re-run refused requests on another model.
- Claude Opus 4.8 launched, supporting mid-conversation system messages, a `high` default effort parameter, lower minimum cacheable prompt length (1,024 tokens), and research-preview fast mode.
- Claude Opus 4.1 deprecation was announced ahead of its scheduled retirement.
- The Messages API response now reports `usage.output_tokens_details.thinking_tokens`, breaking out billed thinking tokens.
- Requests that return `stop_reason: "refusal"` with no generated output are no longer billed.
- Cache diagnostics launched in beta, reporting a `cache_miss_reason` when a prompt's cached prefix diverges from the previous turn.
- Claude Platform on AWS launched, exposing the full Messages/Files/Batches API and Managed Agents through native AWS endpoints with IAM auth.
- Workload Identity Federation reached general availability for authenticating workloads with short-lived OIDC tokens.

## SDK Changes
- The Python, TypeScript, Go, Java, Ruby, PHP, and C# SDKs added support for the persistent-REPL code execution tool version required for programmatic tool calling.
- Those same SDKs now send the `agent-memory-2026-07-22` beta header by default on memory store calls instead of the older managed-agents header.
- A beta Swift package was released adding Claude as a server-side model in Apple's Foundation Models framework across Apple platforms.

## Bug Fixes
- Access to Claude Fable 5 and Claude Mythos 5 was restored after a prior outage.
