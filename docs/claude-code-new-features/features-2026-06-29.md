# Claude Code — What's New (2026-06-29)

## New Features

- Added `claude mcp login <name>` and `claude mcp logout <name>` CLI commands for managing MCP server authentication.
- Added live file path autocomplete when using bash mode (`!`) in the prompt.
- Added `/rewind` command to resume a conversation from before a `/clear` was run.
- Added `autoMode.classifyAllShell` setting to route all Bash/PowerShell commands through the auto-mode classifier.
- Added `sandbox.credentials` setting to prevent sandboxed commands from reading credential files.
- Added `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` environment variable to disable mouse interactions in fullscreen mode.
- Added `claude_code.assistant_response` OpenTelemetry log event containing the model's response text.
- Added auto-mode denial reasons to the transcript, toast notifications, and `/permissions` recent-denials view.
- Added startup notice when MCP servers require authentication before use.
- Added automatic memory-pressure reaping for idle background shell commands.
- Added a "Skills" section to the `/plugin` Installed tab for easier discovery.
- Added status filtering to the `/workflows` agent detail view.
- Added mouse click support for selecting menus in fullscreen mode.

## Improvements

- Reduced streaming CPU usage by ~37% through text update coalescing.
- Reduced long-session memory growth from the terminal output cache.
- Improved MCP server reliability with automatic retries on transient network errors.
- Improved MCP `headersHelper` auth to automatically reconnect on 401/403 responses.
- Improved MCP OAuth with retry logic and better headless environment handling.
- Improved Linux voice mode to distinguish between a missing microphone and a missing SoX installation.
- Improved `claude agents` completed list to better utilize available terminal space.
- Improved Remote session startup with a provisioning checklist display.
- Improved `/plugin` to surface unused plugins for cleanup.
- Improved `/btw` with arrow-key navigation to step through earlier answers.
- Improved vim mode prompt-history search with helpful hints.
- Sandbox network permission dialog now remembers allowed hosts for the session.
- Plugin auto-rename now follows marketplace `renames` maps.

## Bug Fixes

- Fixed hook matchers with hyphenated identifiers incorrectly using substring matching instead of exact matches.
- Fixed voice dictation on macOS capturing silence in extended sessions after input device changes.
- Fixed voice dictation auto-submit for space-free languages (Japanese, Chinese, Thai).
- Fixed background agents resurrecting after being explicitly stopped.
- Fixed background jobs getting stuck indefinitely in "working" state without structured output.
- Fixed background tasks showing blank screens for up to 5 seconds after a crash recovery.
- Fixed background agent daemons becoming unreachable when the control socket fails.
- Fixed `/resume` failing with "No conversation found" for runs that produced no model turns.
- Fixed `--json-schema` structured output allowing the model to call `StructuredOutput` indefinitely.
- Fixed remote MCP tool calls hanging for 5 minutes now aborting with an error.
- Fixed Remote sessions taking ~2.7s longer to start after agent proxy CA install.
- Fixed pasted Korean/CJK text turning into mojibake in terminals using per-byte extended-key events.
- Fixed `/update` hanging over Remote Control with a startup trust dialog.
- Fixed hooks with comma-separated matchers silently never firing.
- Fixed `/permissions` Recently-denied tab approvals not persisting on close.
- Fixed `claude agents` sending slash commands as prompt text instead of showing hints.
- Fixed scroll position jumping during streaming responses while reading earlier output.
