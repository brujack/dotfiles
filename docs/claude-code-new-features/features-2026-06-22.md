# Claude Code — What's New (2026-06-22)

## New Features

- **Auto mode safety guards**: Destructive git commands (`reset --hard`, `checkout -- .`, `clean -fd`, `stash drop`) are now blocked unless you explicitly asked to discard local work; `git commit --amend` is blocked if the commit wasn't made by the agent this session; `terraform destroy` / `pulumi destroy` / `cdk destroy` require explicit stack targeting.
- **`/config key=value` syntax**: Set any setting directly from the prompt (e.g. `/config thinking=false`) — works in interactive, `-p`, and Remote Control modes. `/config --help` lists all available shorthand keys.
- **`attribution.sessionUrl` setting**: Opt out of including the claude.ai session link in commits and PRs generated by web and Remote Control sessions.
- **Agent teams simplified**: `TeamCreate`/`TeamDelete` tools removed; every session with `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1` now has one implicit team — spawn teammates directly with the `Agent` tool's `name` parameter.
- **`Tool(param:value)` permission rules**: Match on a tool's input parameters with wildcard support (e.g. `Agent(model:opus)` to block Opus subagents).
- **`sandbox.allowAppleEvents`**: New opt-in macOS setting lets sandboxed commands send Apple Events.
- **`CLAUDE_CLIENT_PRESENCE_FILE`**: Point at a marker file to suppress mobile push notifications while you're at your machine.
- **Nested `.claude/skills` directories**: Skills in subdirectories now load when working nearby; directory-qualified names (`<dir>:<name>`) prevent collisions.

## Improvements

- **Stream-stall hint**: Message updated to "Waiting for API response · will retry in …" and now triggers after 20 seconds of silence (was 10s).
- **Subagent panel**: Idle subagents auto-hide after 30s, list caps at 5 rows with scroll hints, keyboard hints appear in the footer.
- **Streaming long paragraphs**: Text now renders line-by-line instead of waiting for the first line break.
- **Auto-retry mid-thinking**: API connection drops during a thinking block now retry automatically.
- **Upgraded Bun runtime** to 1.4.
- **`/doctor`** redesigned with a consistent flat tree layout and clearer section status icons.
- **Startup performance**: Fixed a ~120ms-per-launch regression in fresh environments introduced in v2.1.169.

## Bug Fixes

- Fixed `thinking.disabled.display` 400 errors on subagent spawns and session-title generation.
- Fixed WebSearch returning empty results in subagents.
- Fixed fullscreen TUI corruption (statusline mid-screen, duplicated spinner rows) in Windows Terminal under heavy nested-subagent load.
- Fixed turns silently completing with no visible output when the model returned only a thinking block.
- Fixed Write/Edit producing 0-byte or truncated files on network drives and cloud-synced folders.
- Fixed mid-stream connection drops losing partial responses.
- Fixed mouse-wheel scrolling in WSL2 under Windows Terminal and VS Code (regression from v2.1.172).
- Fixed startup blocking up to 15 seconds when account settings fetch is slow on a degraded network.
- Fixed foreground subagents spawning unbounded nested chains — now respect the 5-level depth limit.
- Fixed Linux sandbox failing to start when `.claude/settings.json` or `.claude/skills` is a symlink.
